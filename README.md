# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

# Description

Where's Waldo game build on React. Player'll start with a large photograph contains several characters that player need to find. Player will make selections for each character and get feedback whether correct or not

# Tools

Project is written in React with Typescript.
Character's data stored on Firestore.
